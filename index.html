GUÍA DE CLONADO DE IMÁGENES

[ SECCIÓN 1: PORTADA Y REQUISITOS ]

Materiales

  • 1 Disco con conexión USB con hirens (preferible dentro de ventoy)
  • 2 Tener 1 Equipo al cual se le realizará ISO (sin ser custom)
  • 3 Disco de destino (Será el disco que reemplazará al de fabrica)
  • 4 Adaptador (es) de ssd m.2 (preferiblemente para mas velocidad de clonado en la ISO final)

Boot menu de las laptops

DELL F12
LENOVO F12, (FN+F12)
ASUS F8
HP F9
MSI F11

Una vez teniendo todo lo anterior mencionado se puede comenzar.

[ SECCIÓN 2: INICIO DEL PROCESO ]

Proceso a realizar 1 vez para sacar el ISO

hay conectar el USB con hirens, iniciar el equipo y acceder al boot menu, la mayoria de equipos se bloqueara debido al secure boot, hay que seleccionar la key de Ventoy "Enroll This key" para poder acceder sin quitar el Secure boot, de ahí meterse directamente a Hirens boot, dentro de hirens hay que buscar Macrium.

[INSERTE AQUÍ IMAGEN DE: Icono o acceso a Macrium Reflect dentro del escritorio de Hiren's Boot]

[ SECCIÓN 3: Errores puntuales ]

Error de Detección de Disco en ensambles (Modo AHCI)

(Esto no ocurre en las laptop) Solamente aplica si se Clonan ISO de ensambles que presenten el error, si al iniciar Macrium Reflect no se visualiza el disco del ensamble, sigue estos pasos:

Problema

El controlador está en modo RAID o Intel VMD (común en procesadores nuevos).

Solución

Entra a la BIOS y cambia el modo de almacenamiento a AHСІ.

Efecto

Esto permitirá que el entorno de Hiren's reconozca discos NVMe de forma nativa sin necesidad de cargar drivers externos.

[INSERTE AQUÍ IMAGEN DE: Opción de cambio de RAID/VMD a AHCI en la interfaz de una BIOS]

[ SECCIÓN 4: CREACIÓN DE LA IMAGEN MÁSTER ]

Generación de la Imagen a propagar (.mrimg)

dentro de macrium hay que buscar la opción "Backup" dentro de la opción backup, seleccionamos el disco que se desea clonar, en este caso el disco 1, y todas sus particiones de fábrica.

[INSERTE AQUÍ IMAGEN DE: Menú superior de Macrium Reflect con la pestaña "Backup" e "image this disk" seleccionada]

[INSERTE AQUÍ IMAGEN DE: Opción "Carpeta del iso guardar" en la interfaz de Macrium]

Hay que usar la opción “Image this disk…” dentro, solicitara donde alojar esa ISO, lo alojamos en las carpetas de “ISO CUSTOM” La ISO se guardará con formato “.mrimg” hay que nombrar al archivo del ISO con el SKU del producto, en el ejemplo es el ISO del SKU: “542096”

[ SECCIÓN 5: PREPARACION DE ISO ]

Preparacion del entorno para clonar el ISO

Una vez está guardada esta ISO, procedemos a clonarla en un disco de mayor capacidad. Primero, instalamos el disco de mayor capacidad dentro del equipo (de preferencia un adaptador de m.2 hacia USB C o USB 3.0 para mayor velocidad de clonado) Hay que ir a la opcion de “Restore”, dentro hay que usar la opcion “Browse for an image file…”

[INSERTE AQUÍ IMAGEN DE: Menú superior de Macrium Reflect con la pestaña "Restore" y "Browse for an image file..." seleccionada]

[INSERTE AQUÍ IMAGEN DE: Opción "Select an image file" en la interfaz de Macrium]

Dentro, seleccionamos el ISO necesario para clonarlo, en este caso, el del ejemplo “ISO 542096-00-00.mrimg”. que es el que corresponde al ISO 542096

[ SECCIÓN 5: RESTAURACIÓN Y AJUSTE DE PARTICIONES ]

Procedimiento de Restauración

Dentro verificamos que las particiones del ISO son correctas y seleccionamos “Restore Image”

[INSERTE AQUÍ IMAGEN DE: Menu de Macrium Reflect con la pestaña "Restore image"]

[INSERTE AQUÍ IMAGEN DE: Opción "Select a disk to restore" en la interfaz de Macrium]

Dentro de “Restore image” hay que seleccionar el disco vacío donde se busca clonar el ISO original del fabricante para hacerlo el ISO de 1 tb que se va a clonar masivamente.

ADVERTENCIA CRÍTICA

NO utilizar el botón 'Copy selected partitions' de forma directa si el disco de destino es más grande que el origen, ya que esto bloqueará el espacio sobrante como 'No asignado' y no se podrá expandir después fácilmente. Si se agrega todo de golpe quedará más de la mitad del espacio vacío sin posibilidad de ampliar ni usar ese espacio en blanco.

Pasos para el ajustarlo manual y evitar errores:

01

Arrastre las primeras 3 particiones al disco de destino.

02

Antes de arrastrar la última partición (WINRE DRV), seleccione la partición 3-Windows-SSD.

03

Haga clic en "Restored Partition Properties".

[INSERTE AQUÍ IMAGEN DE: Botón "Restored Partition Properties" resaltado debajo del mapa de particiones]

04

En el apartado de "Free Space", ajuste el valor manualmente para dejar exactamente el espacio que requiere la partición de recuperación (ej. 1.95 GB o 2000 MB según el origen).

05

Una vez ajustado, arrastre la partición 4-WINRE DRV al espacio final.

[ SECCIÓN 6: NOTAS DE PRODUCCIÓN ]

Una vez aquí, hemos finalizado el clonado del ISO del fabricante dentro de un Disco de 1TB, de aquí en adelante, hay que sacarle ISO a ese disco de 1TB con el sistema del fabricante, lo podemos propagar a los demás discos sin necesidad de hacer la expansión del espacio a cada disco, solamente copiando el ISO y dándole a Siguiente.

Nota de Producción

Para los equipos restantes, NO es necesario repetir el ajuste de particiones. Al usar la Imagen Máster de 1TB sobre discos de 1TB, el volcado es directo (Full Disk Restore).

[ SECCIÓN 7: VALIDACIÓN FINAL "Opcional" ]

Validación del Entorno de Recuperación (WinRE)

Una vez finalizado el primer clonado del lote, es recomendable validar que la partición de recuperación haya quedado configurada correctamente. Para ello, abra una consola de comandos (CMD) como administrador y ejecute:

C:\Windows\System32> reagentc /info

Verifique que el estado indique "Enabled".
